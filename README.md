# Evolução em Matemática

Demo:

Projeto de estudo em **HTML, CSS e JavaScript puro** para registro e acompanhamento da evolução de alunos em matemática.

A aplicação simula um pequeno sistema escolar, permitindo cadastrar alunos, visualizar detalhes individuais e registrar a evolução de aprendizagem por meio de uma interface simples e organizada.

## Funcionalidades atuais
- Cadastro de alunos
- Listagem de alunos
- Visualização de detalhes do aluno
- Uso de modais com a tag `<dialog>`
- Layout responsivo
- Organização de arquivos e assets

## Funcionalidades em desenvolvimento
- Edição de alunos
- Remoção de alunos
- Registro dinâmico do histórico de evolução
- Persistência de dados

## Tecnologias utilizadas
- HTML5
- CSS3
- JavaScript (DOM, eventos e manipulação de elementos)

## Objetivo do projeto
Projeto desenvolvido com fins **educacionais**, focado no aprendizado de:
- Estruturação de interfaces
- Manipulação do DOM
- Organização de código front-end
- Boas práticas iniciais em JavaScript
